# ✨ Features

- AI-based lost & found item matching
- Smart text similarity detection
- Image similarity matching using OpenCV
- User authentication (Login/Register)
- Secure private messaging system
- Notifications system
- Admin dashboard
- Leaderboard & reward system
- Mobile responsive modern UI
- Dark/Light theme support

# 🧠 AI Matching Engine

Campus Lost & Found AI uses:

- TF-IDF cosine similarity
- Jaccard similarity
- ORB feature matching
- Histogram comparison
- Perceptual hashing

Final Match Score:
- Text similarity → 65%
- Image similarity → 35%
